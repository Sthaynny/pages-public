Política de Privacidade — Cura.li / Cura.li+

Última atualização: 11 de julho de 2026
Contato: igorsthaynny@gmail.com

Este documento descreve como o aplicativo trata dados no dispositivo. Aplica-se às edições Cura.li e Cura.li+.

1. Resumo

PerguntaResposta
Coletamos dados de saúde/medicação? Sim — inseridos por você no aparelho
Os dados saem do dispositivo no uso normal? Não (armazenamento local SQLite / preferências)
Vendemos ou usamos dados para anúncios? Não
Conta de usuário / login? Não no MVP

2. Dados tratados

No dispositivo

  • Medicamentos, dosagens, horários e validade
  • Histórico de doses (tomada, pulada, atrasada)
  • Dados do paciente (nome, idade, alergias) que você cadastrar
  • Preferências do app (tema, alertas)

Não coletamos (MVP)

  • Conta, e-mail de login ou telefone (além do e-mail que você usar para suporte)
  • Localização, contatos, fotos, microfone ou câmera
  • Dados financeiros ou prontuário clínico

3. Finalidade

Organizar lembretes de medicação e apoiar o cuidador. O app não substitui orientação médica — ver os Termos de uso.

4. Compartilhamento e terceiros

  • Não compartilhamos dados de medicação com anunciantes.
  • Exportação de histórico ou backup (Cura.li+) só ocorre quando você escolhe compartilhar/salvar o arquivo pelo sistema do aparelho.
  • Relatórios de falha (crash): em builds release com Firebase Crashlytics, podemos receber dados técnicos do dispositivo (modelo, SO, stack trace), sem conteúdo de medicamentos.

5. Retenção e exclusão

  • Dados permanecem no aparelho até você excluí-los no app ou desinstalar o aplicativo.
  • Desinstalar remove o banco local e as preferências.
  • Backups que você exportou ficam sob seu controle (arquivos fora do app).

6. Crianças

O app é voltado a cuidadores. Não coletamos intencionalmente dados de crianças para marketing. Dados de pacientes menores só existem se o cuidador os cadastrar localmente.

7. Alterações

Podemos atualizar esta política. A data no topo indica a versão vigente. Mudanças relevantes serão refletidas também nas fichas das lojas.

8. Contato
